# 《燕云十六声》装备调律系统完整说明文档


## 一、系统基础规则
装备共包含 **5个词条槽位**，词条的获取与调整遵循以下核心逻辑：
1. **宫槽**：装备获取时自带的**初始词条**，不可更改，决定装备的基础属性基底；
2. **商/角/徵/羽槽**：通过“调律”功能解锁，从对应**装备部位+流派**的转律库中选择词条附加；
3. **定音**：完成商/角/徵/羽4个槽位的调律后解锁，可附加1个定音词条（主副武器/环/佩用通用定音库；头/胸/胫/腕甲用流派专属定音库）。


## 二、初始词条库（宫槽）
宫槽词条为装备固有属性，不可调整，不同部位的初始词条池如下：

| 装备部位  | 初始词条内容                                                                 |
| ----- | --------------------------- |
| 武器    | 最大外功、最小外功、最大无相攻击、最小无相攻击、敏、势                       |
| 环/佩   | 最大外功、最小外功                                                           |
| 头/胸   | 会心率、会意率、精准率、最大气血、外功防御                                   |
| 胫甲/腕甲 | 会心率、会意率、劲、精准率、体、御、最大气血、外功防御                       |


## 三、转律库（商/角/徵/羽槽）
调律时需从对应**装备类型+流派**的转律库中选择词条，各流派转律属性及数值区间如下：

### 3.1 主副武器转律库
仅适用于主副武器，对应流派为鸣金流派：

| 流派     | 对应词条 | 转律属性效果                                                                 |
|----------|----------|------------------------------------------------------------------------------|
| 鸣金流派 | 虹、影   | - 核心属性：最大外功攻击（+45.3~90.6）、最大无相攻击（+25.7~51.4）、会心率（+5.2%~10.4%）、会意率（+2.6%~5.2%）<br>- 基础属性：势（+28.7~57.4）、劲（+28.7~57.4） |

| 装备类型     | 定音词条来源  | 定音词条内容                                          |
| -------- | ------- | ----------------------------------------------- |
| 主副武器、环、佩 | 通用定音库   | 无相穿透（+9.1~15.2）、外功穿透（+7.7~12.8）、外功抗性（+7.7~12.8） |
