# 《燕云十六声》核心战斗属性概念及数值文档


## 1. 精准率
- **定义**：攻击时精准命中目标的几率。
- **核心效果**：
  - 精准命中：有几率触发**会心、会意**；
  - 未精准命中：有几率触发**擦伤、会意**（擦伤基于自身攻击最小值结算）。
- **生效规则**：与81级玩家/怪物对抗时，超出基础精准率的部分会受**判定抗性**影响而降低生效值。
- **当前数值**：
  - 面板精准率：115.0%
  - 同等级段对抗生效精准率：92.0%
- **获取途径**：单人模式等级突破、装备调律


## 2. 会心率
- **定义**：攻击时触发会心的概率。
- **核心效果**：会心触发时，**伤害效果+50.0%、治疗效果+50.0%**。
- **生效规则**：
  - 需先精准命中才能触发；
  - 与81级玩家/怪物对抗时，受判定抗性影响降低生效值，且**生效会心率上限为80%**。
- **当前数值**：
  - 面板会心率：132.2%
  - 同等级段对抗生效会心率：71.5%
- **获取途径**：敏属性转化、装备调律


## 3. 会意率
- **定义**：攻击时触发会意的概率。
- **核心效果**：会意触发时，**伤害效果+35.0%**，且必定基于自身攻击最大值结算。
- **生效规则**：
  - 触发不受精准概率影响；
  - 与81级玩家/怪物对抗时，受判定抗性影响降低生效值，且**生效会意率上限为40%**。
- **当前数值**：
  - 面板会意率：21.0%
  - 同等级段对抗生效会意率：11.3%
- **获取途径**：势属性转化、装备调律


## 4. 直接会心率
- **定义**：额外提升攻击时触发会心的几率。
- **核心效果**：不受**会心率上限**限制，也不会被目标的判定抗性影响。
- **当前数值**：
  - 直接会心率：9.2%
  - 最终会心率（面板+直接）：132.2%+9.2%
  - 同等级对抗最终会心率（生效+直接）：71.5%+9.2%
- **获取途径**：心法突破


## 5. 直接会意率
- **定义**：额外提升攻击时触发会意的几率。
- **核心效果**：不受**会意率上限**限制，也不会被目标的判定抗性影响。
- **当前数值**：
  - 直接会意率：0.0%
  - 最终会意率（面板+直接）：21.0%+0.0%
  - 同等级对抗最终会意率（生效+直接）：11.3%+0.0%
- **获取途径**：心法突破


## 6. 擦伤转化率
- **定义**：攻击即将造成擦伤时，将该次伤害转化为普通伤害的几率。
- **当前数值**：0.0%


## 7. 判定抗性
- **定义**：角色/怪物等级达到81级后获得的属性。
- **核心效果**：降低被攻击时**精准、会心、会意**的命中概率，同时降低被治疗时的会心概率。
- **等级段对应抗性**：
  - 十二级·无为（81-85级）：15%
  - 十三级·普观（86-90级）：30%
  - 十四/十五级（91-95级）：45%
  - 十六/十七级（96-100级）：65%
  - 十八/十九级（100级）：85%
- **当前数值**：85.0%（对应十八/十九级）


## 8. 全部武学增效
- **定义**：使用任意武学技能时的增伤/增疗效果。
- **核心效果**：提升所有武学技能的伤害/治疗（包括基础技能、持续性效果），**对奇术、心法无效**。
- **生效规则**：多个效果可叠加，且可与“指定武学增效”叠加。
- **当前数值**：3.2%
- **获取途径**：装备调律-环佩


## 9. 指定武学增效
- **定义**：使用指定武器的武学技能时的增伤/增疗效果。
- **核心效果**：提升对应武器武学技能的伤害/治疗（包括基础技能、持续性效果），**对奇术、心法无效**。
- **生效规则**：多个同武器的效果仅生效数值最大的一个。
- **当前数值**：
  - 总面板指定武学增效：7.4%
  - 细分武器增效：陌刀武学增伤7.4%（其余武器为0.0%）
- **获取途径**：对应武器的装备调律


## 10. 对首领单位增伤
- **定义**：对首领类型单位造成的所有伤害加成。
- **核心效果**：提升对首领单位的全伤害，多个效果可叠加。
- **当前数值**：6.1%
- **获取途径**：装备调律-胫甲和腕甲
