# 《燕云十六声》流派分类与装备词条系统详解
## 一、武学流派分类及核心属性倾向
### 1. 流派体系划分
| 大流派   | 细分流派       |
|----------|----------------|
| 鸣金     | 虹、影         |
| 裂石     | 威、钧         |
| 破竹     | 风、尘、鸢     |
| 牵丝     | 玉、霖         |

### 2. 核心属性与定位说明
- **属性倾向**：仅「鸣金流派」需追求**会意率**（接近上限40%）；其余所有流派均为「会心流派」，核心堆**会心率**（上限80%）。
- **特殊定位**：牵丝·霖为唯一奶妈流派，需侧重生存与治疗相关属性。

## 二、装备词条槽位规则
### 1. 槽位构成（共6个）
| 槽位名称 | 解锁/操作规则                                                                 |
|----------|------------------------------------------------------------------------------|
| 宫       | 装备自带固定词条，**玩家无法更改**                                           |
| 商、角、徵、羽 | 可通过「调律」从对应词条库抽取词条嵌入；支持「转律」（单个装备仅可转律1次）重新抽取 |
| 定音     | 需完成商、角、徵、羽五音槽位调律后解锁，从定音词条库抽取词条嵌入             |

## 三、转律系统核心规则
1. 适用范围：仅商、角、徵、羽四个槽位可转律，定音槽位不可转律。
2. 操作限制：**单个装备仅能选择1个槽位进行1次转律**，转律后从对应流派的「转律词条库」重新抽取词条。
3. 核心作用：解决初次调律词条无用的问题，精准匹配流派需求（如鸣金流派转律后可锁定会意率相关词条）。
